หุ้นไทย  เปิดตลาดบวก 7.81 จุด ทะลุ 1,800 จุด เพิ่มขึ้น 380 หลักทรัพย์ นักวิเคราะห์ ชี้ แรงซื้อจากนักลงทุนสถาบันฯ ยังหนาแน่น
บรรยากาศการลงทุนในตลาดหุ้นไทยเช้าวันนี้ (8 ม.ค.) เปิดทำการเมื่อเวลา 10.00 น. ดัชนีปรับตัวเพิ่มขึ้น 7.81 จุด แตะที่ระดับ 1,803.26 จุด ด้วยมูลค่าการซื้อขาย 3,817.14 ล้านบาท โดยหลักทรัพย์เพิ่มขึ้น 380 หลักทรัพย์ ลดลง 276 หลักทรัพย์ ไม่เปลี่ยนแปลง 350 หลักทรัพย์

นักวิเคราะห์ บล.เคจีไอ คาด SET วันจันทร์ขึ้นต่อ หลังปัจจัยภายนอกยังเป็นบวก รวมถึงแรงซื้อจากนักลงทุนสถาบันฯ ยังหนาแน่น อย่างไรก็ดี SET ถึงเป้าไตรมาส 1 ปี 2561 แล้วและน่าจะผันผวนมากขึ้นในสัปดาห์นี้ และมองอัพไซด์ระยะสั้นของดัชนีฯ มีจำกัด แนะกลยุทธ์เพิ่มหุ้นขนาดกลาง-เล็กที่ธีมการลงทุนน่าสนใจ เข้าพอร์ต
ด้าน บล.เอเอสแอล มองว่า SET Index สัปดาห์แรกของปีปรับตัวขึ้นแรง เป็นไปตามทิศทางของตลาดภูมิภาคและตลาดโลกที่ปรับตัวขึ้นทำสถิติ เช่นเดียวกันจากตัวเลขเศรษฐกิจที่ยังเร่งตัวขึ้น และราคาสินค้าโภคภัณฑ์ที่ยังอยู่ในระดับสูง
