บรรยากาศการลงทุนในตลาดหุ้นไทยเช้าวันนี้ (10 ม.ค.) เปิดทำการเมื่อเวลา 10.00 น. ดัชนีปรับตัวเพิ่มขึ้น 7.93 จุด แตะที่ระดับ 1,803.14 จุด ด้วยมูลค่าการซื้อขาย2,566.33 ล้านบาท โดยหลักทรัพย์เพิ่มขึ้น 356 หลักทรัพย์ ลดลง 116 หลักทรัพย์ ไม่เปลี่ยนแปลง 266 หลักทรัพย์

นักวิเคราะห์ บล.คันทรี่ กรุ๊ป คาดว่า SET แกว่งตัวเชิงบวกในกรอบ 1,791 – 1,802 จุดหลังสัญญาน้ำมันดิบ WTI ปิดตัวที่ระดับ 62.96 ดอลลาร์/บาร์เรล อันเป็นปัจจัยสนับสนุนเชิงบวกต่อกลุ่มพลังงาน
ขณะที่ บล.เคจีไอ คาด SET วันพุธไซด์เวย์ต่อ และความเสี่ยงทางลงรอบนี้ มองไม่ลึก ที่ 1,760 จุด อีกทั้งขณะนี้ งหุ้นใหญ่ในกลุ่มพลังงานและปิโตรฯ ยังโดดเด่นในช่วงนี้ ตามราคาน้ำมันดิบที่พุ่งต่อ ทำจุดสูงสุดใหม่รอบกว่า 3 ปี
ด้าน บล.ยูโอบีเคย์เฮียน เปิดเผยว่า ล่าสุด ธนาคารโลก (world bank) ปรับเพิ่มคาดการณ์การเติบโตของเศรษฐกิจโลกปี 61 ขึ้นสู่ระดับ 3.1% จากเดิมคาด 2.8 พร้อมระบุว่า เศรษฐกิจทั่วโลกมีแนวโน้มฟื้นตัวเป็นวงกว้าง โดยได้ปัจจัยหนุนจากการค้าและการลงทุนที่ปรับตัวดีขึ้น
