บริษัทหลักทรัพย์ กสิกรไทย จำกัด (มหาชน) ระบุบทวิเคราะห์วันนี้ (30เม.ย.) ว่า  หุ้นไทย  สัปดาห์นี้ตลาดจะให้น้ำหนักไปที่ปัจจัยต่างประเทศ ซึ่งจะมีการรายงานตัวเลขเศรษฐกิจชุกชุม รวมไปถึงการประชุม FOMC คาดว่าตลาดจะตอบรับในทางบวก และในช่วงปลายสัปดาห์คาดว่าจะเริ่มมีแรงซื้อเพื่อเก็งกำไรผลประกอบการบริษัทจดทะเบียนที่จะมีการประกาศในช่วง 7-11 พ.ค.กว่า 290 บริษัท
สำหรับปัจจัยในประเทศมีอิทธิพลน้อย คาดตลาดเคลื่อนไหวตามรายงานเศรษฐกิจต่างประเทศ การรายงานผลประกอบการบริษัทจดทะเบียนมีเพียง 15 บริษัทเท่านั้น ทำให้ปัจจัยต่างประเทศจะมีน้ำหนักในการกำหนดทิศทางของตลาดมากกว่า

ระหว่างสัปดาห์ SET Index จะเคลื่อนไหวตาม Sentiment ต่างประเทศ ซึ่งจะมีการรายงานเศรษฐกิจที่ต้องติดตามไล่ตั้งแต่ 30 เม.ย. เงินเฟ้อพื้นฐานของสหรัฐ (Core PCE) ตลาดคาดไว้ที่ 1.9% คาดว่าแรงซื้อดังกล่าวจะช่วยทำให้ SET Index ขยับขึ้นไปทดสอบ 1,790 ถึง 1,800 จุดได้
ระยะสั้นเน้นหุ้นที่จะมีผลประกอบการแข็งแกร่ง ตลาดกำลังอยู่ในช่วงขาดปัจจัยบวกใหม่ๆ หุ้นขนาดกลางและเล็กส่วนใหญ่ปรับลดลงอย่างรุนแรงในช่วงที่ผ่านมา อย่างไรก็ตามยังมีหุ้นหลายๆ ตัวสามารถเคลื่อนไหวสวนตลาดได้ ซึ่งส่วนใหญ่จะเป็นหุ้นที่มีแนวโน้มผลประกอบการในงวด 1Q61 แข็งแกร่ง ระยะสั้นจึงให้เน้นลงทุนหุ้นงบเด่นเลือก SPALI MEGA BGRIM เป็น Top Pick
กลยุทธ์การลงทุน คาดแรงขายพลังงานจะเริ่มเบาลง ขณะที่ตลาดมีแรงซื้อคืนกลุ่มธนาคาร และแรงซื้อเพื่อเก็งผลประกอบการ คาดว่าจะหนุนให้ดัชนีฟื้นกลับไปทดสอบ 1,785 จุดในวันนี้ เลือก SPALI MEGA BGRIM เป็นหุ้นเด่น ส่วนในภาพระยะกลาง-ยาว แนะให้ซื้อสะสมหุ้นใน
