บริษัทหลักทรัพย์ ฟินันเซีย ไซรัส จำกัด (มหาชน) ระบุบทวิเคราะห์วันนี้ (24 ก.ย.) ว่า แนวโน้มตลาดหุ้นไทย SET Index คาดว่าจะแกว่งตัว Sideways หลังจากที่ปรับตัวขึ้นกว่า 80 จุดในช่วงสัปดาห์เศษๆที่ผ่านมา และเริ่มเห็นแรงขายทำกำไรออกมาในช่วงท้ายตลาดในระยะหลัง

โดยวันนี้สหรัฐฯ-จีนจะเริ่มเก็บภาษีสินค้านำเข้าระหว่างกันรอบใหม่ ขณะที่กระแสเงินทุนต่างชาติคาดว่าจะชะลอตัวเพื่อติดตามการประชุม FOMC กลางสัปดาห์ซึ่งค่อนข้างแน่นอนว่าจะขึ้นดอกเบี้ยเป็น 2.25%
อย่างไรก็ตามจากปัจจัยในประเทศที่ยังแกร่งทั้งเศรษฐกิจและการเมือง ทำให้ระยะกลาง-ยาวดัชนียังมีแนวโน้มปรับตัวขึ้นต่อ ซึ่งเรายังเน้นลงทุนในหุ้นกลุ่ม Domestic Play โดยเลือกตัวที่ Laggard มากขึ้น
สำหรับกลยุทธ์ เน้นลงทุนในกลุ่ม Domestic และ Laggard Play ระยะกลางยาวยังเน้นถือ Let Profit Run ขณะที่หุ้นเด่นเดือน ก.ย. ได้แก่ ASK, CHG, CK, CPALL, PRM
