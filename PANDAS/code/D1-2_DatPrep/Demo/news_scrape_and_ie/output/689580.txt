บริษัทหลักทรัพย์ ฟินันเซีย ไซรัส จำกัด (มหาชน) ระบุบทวิเคราะห์วันนี้ (29 พ.ย.) ว่า แนวโน้มตลาดหุ้นไทย เราคาดว่า SET Index จะปรับตัวขึ้นทดสอบระดับ 1,650 จุด จากบรรยากาศการลงทุนที่สดใส โดยนักลงทุนผ่อนคลายมาก 
ขึ้นจากถ้อยแถลงของประธาน FED ที่บอกว่าอัตราดอกเบี้ยในปัจจุบันต่ำกว่าระดับ Neutral เพียงเล็กน้อย ซึ่งต่างจากที่เคยพูดเมื่อเดือน ต.ค.

อย่างไรก็ตาม ยังมองว่าการเจรจาการค้าระหว่างทรัมป์กับ สี จิ้น ผิง ในการประชุม G20 ปลายสัปดาห์นี้ยังเป็นปัจจัยสำคัญและชี้นำทิศทางตลาด โดยล่าสุดมีกระแสว่าทรัมป์กังวลต่อผลกระทบจากสงครามการค้าที่ลากยาว ทำให้มีโอกาสที่จะประนีประนอมกับจีนเพื่อหาทางออกร่วมกัน
สำหรับกลยุทธ์การลงทุนหุ้นไทย ส่วนที่ซื้อแล้วถือ Let Profit Run ต่อ ทยอยสะสมเพิ่มเมื่ออ่อนตัวลงใกล้เคียงระดับ 1,600 จุด ขณะที่หุ้นเด่นเดือน พ.ย. ได้แก่ CHG, CK, PRM, PTTEP, TISCO

(+) เฟดส่งสัญญาณไม่เร่งขึ้นดอกเบี้ย ดาวโจนส์บวกกว่า 600 จุดเมื่อคืน หลังประธานเฟดส่งสัญญาณไม่เร่งขึ้นดอกเบี้ย โดยระบุว่าอัตราดอกเบี้ยจะยังอยู่ในระดับต่ำกว่าค่ามาตรฐาน แต่ก็ต่ำกว่าเป้าหมายที่จะทำให้เศรษฐกิจเป็น neutral เพียงเล็กน้อย
ความหมายคือดอกเบี้ยระดับปัจจุบันไม่ได้ช่วยเร่งหรือดึงรั้งเศรษฐกิจ (สมดุลหรือใกล้สมดุลแล้ว) ทั้งนี้ตลาดยังคาดว่าเฟดจะขึ้นดอกเบี้ยในเดือนหน้า 0.25% แต่ปี 2019 อาจขึ้นเพียง 1 ครั้งจากก่อนหน้านี้ที่คาดขึ้น 3 ครั้ง
(-) ราคาน้ำมันดิบร่วง โดย NYMEX ส่งมอบเดือน ม.ค. ลดลง -1.27 ดอลลาร์/บาร์เรล (-2.5%) อยู่ที่ 50.29 ดอลลาร์/บาร์เรล หลังตัวเลขสต็อคน้ำมันดิบของสหรัฐเพิ่มขึ้นมากกว่าคาดเกือบ 3 ล้านบาร์เรล ส่วนค่าการกลั่นสิงคโปร์ปรับตัวลง -19% มาอยู่ที่ 3.59 ดอลลาร์/บาร์เรล เป็นลบต่อ PTTEP TOP SPRC แต่เป็นบวกกับ SCC AOT และ AAV
