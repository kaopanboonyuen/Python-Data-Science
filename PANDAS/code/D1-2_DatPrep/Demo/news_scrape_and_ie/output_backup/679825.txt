บริษัทหลักทรัพย์ ฟินันเซีย ไซรัส จำกัด (มหาชน) ระบุบทวิเคราะห์วันนี้ (18 ต.ค.) ว่า แนวโน้มตลาดหุ้นไทย เราคาด SET Index จะแกว่งตัว Sideways Down โดยกลุ่มพลังงานคาดว่าจะถ่วงตลาดจากจากราคาน้้ามันดิบที่ร่วงแรง ส่วน 
การรายงานการประชุม FED ที่ส่งสัญญาณปรับขึ้นดอกเบี้ยต่อเนื่อง เรามองว่าตลาดซึมซับประเด็นดังกล่าวไปมากแล้ว

ขณะที่ไม่มีประเทศคู่ค่าที่บิดเบือนค่าเงินจากการเปิดเผยข้อมูลของสหรัฐฯ โดยจีนอยู่ใน Watch List ทำให้ตลาดยังไม่มีแรงกดดันเพิ่มเติม ประเด็นที่ต้องจับตายังเป็นเรื่องการเจรจา Brexit และการอนุมัติงบประมาณขาดดุลอิตาลีจาก EU เรายังมีมุมมองเป็นบวกในระยะกลาง-ยาวต่อ SET Index และยังเน้นกลุ่ม Domestic Play เช่นเดิม
สำหรับกลยุทธ์การลงทุน เน้นสะสมหุ้น Domestic ในช่วงลบ และถือลงทุนระยะกลาง-ยาว ขณะที่หุ้นเด่นเดือน ต.ค. ได้แก่ BDMS, CPALL, CPN, MINT, PTTGC
