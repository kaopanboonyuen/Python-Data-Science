บล.ฟินันเซีย ไซรัส จำกัด (มหาชน) ระบุบทวิเคราะห์วันนี้ (27 พ.ย.) ว่า แนวโน้มตลาดหุ้นไทย เราคาดว่า SET Index จะแกว่งตัว Sideways โดยยังอยู่ในช่วงการ Rebound หลังร่วงแรงในช่วงก่อนหน้า ซึ่งวันนี้คาดว่ากลุ่มพลังงานจะปรับตัวได้ดีขึ้นจากราคาน้้ามันดิบที่เริ่มฟื้นตัว รวมถึงอาจเห็นการปรับลดกำลังการผลิตจากกลุ่ม OPEC ในกาประชุมเดือนหน้า

อย่างไรก็ตามปัจจัยสำคัญที่ต้องจับตาคือการประชุม G20 ปลายเดือนนี้ ซึ่งจะมีการพูดคุยนอกรอบระหว่างทรัมป์กับสี จิ้น ผิงในประเด็นสงครามการค้า รวมถึงการโหวตสนับสนุน Brexit ในต้นเดือนหน้า
สำหรับกลยุทธ์การลงทุนหุ้นไทย ทยอยสะสมหุ้นเมื่ออ่อนตัวลงหาระดับ 1,585-1,600 จุด ขณะที่หุ้นเด่นเดือนพ.ย. คือ CHG, CK, PRM, PTTEP, TISCO
