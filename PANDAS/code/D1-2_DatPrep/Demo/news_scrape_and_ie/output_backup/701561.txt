บริษัทหลักทรัพย์ กสิกรไทย จำกัด (มหาชน) ระบุบทวิเคราะห์วันนี้ (17 ม.ค.) ว่า วันนี้หุ้นกลุ่มธนาคารและการเงินหลายตัวจะมีการประกาศงบออกมาประกอบด้วย BBL, KKP, KTB, THANI, TMB, TCAP โดยวานนี้ KKP ปรับตัวเพิ่มขึ้นมา 0.37% เก็งผลประกอบการงวด 4Q/61 ที่น่าจะออกมาเติบโต 4%QoQ และ24%YoY มีสิทธิถูก sell on fact หากงบออกมาไม่ดีกว่าคาด
ส่วน BBL วานนี้ปรับตัวลงเยอะ -1.46% หากผลประกอบการออกมาสอดคล้องกับ consensus คาดว่าจะมีแรงซื้อกลับ โดย consensus คาดการณ์กำไรงวด 4Q/61 ที่ 9,251 ล้านบาท เติบโต 2%QoQ และ 9%YoY

ในขณะที่ BAY ประกาศงบออกมาแล้วโดยกำไรสุทธิไตรมาส 4/61โตลดลง-1.9% QoQ แต่เติบโต 7.5%YoY ต่ำกว่าที่ consensus คาดไว้ว่าจะเติบโต 9%YoY และลดลง -1% QoQ ราคาหุ้นวานนี้ปรับตัวเพิ่มขึ้นมา+0.68% วันนี้มีสิทธิโดนขายแต่ราคาหุ้นเทรดอยู่ที่ระดับล่างอยู่แล้วไม่น่าปรับตัวลงไปเยอะ
SCB รายงานกำไรสุทธิไตรมาส 4/2561 ที่ 7.1 พันลบ. ลดลง 23% YoY และ 33% QoQ ต่ำกว่าประมาณการของ KS research และ consensus เป็นผลจากการตั้งสำรองที่สูงกว่าคาดเพื่อสะท้อนคาดการณ์ความไม่แน่นอนของเศรษฐกิจ
ขณะที่ KBANK รายงานผลประกอบการไตรมาส4/2561 มีกำไรสุทธิ 7.03 พันลบ. -27.82%YoY +12%QoQ ต่ำกว่าที่ consensus คาดการณ์ โดยมองว่าราคาหุ้น KBANK และ SCB จะถูกกดดันจากแรงขาย รวมถึงจะสร้าง sentiment ลบต่อกลุ่มธนาคารที่ยังไม่ได้ประกาศงบออกมา
ทั้งนี้ SET Index ผันผวน ดัชนีกลับมายืนเหนือ 1,580 จุด ได้เม็ดเงินจากต่างชาติเข้ามาช่วยหนุนตลาด วานนี้ SET Index เคลื่อนไหวแดนบวกเกือบทั้งวันและสามารถปิดเหนือ 1,580 จุดได้ ขณะที่ต่างชาติยังเข้าซื้อหุ้นไทยต่อเนื่อง
มองว่าวันนี้ตลาดหุ้นไทยน่าจะฟื้นตัวต่อจากปัจจัยหนุนต่างประเทศที่สหรัฐเตรียมลดภาษีนำเข้าแก่จีน รวมถึงปัจจัยบวกจากในประเทศเรื่องการเลือกตั้งที่มีการส่งสัญญาณออกมาว่าเป็นวันที่ 24 มี.ค. อย่างไรก็ตามดัชนีอาจไปได้ไม่ไกลเนื่องจากถูกแรงกดดันจากกลุ่มธนาคารที่ผลประกอบการต่ำกว่าคาด
สำหรับกลยุทธ์การลงทุนเลือกหุ้น SVI เพื่อเก็งผลประกอบการไตรมาส 4 ที่จะเติบโตจากยอดคำสั่งซื้อที่ยังแข็งแกร่ง ผนวกกับ MEGA ที่ผลประกอบการณ์ไตรมาส 4 จะเป็นจุดสูงสุด และหุ้นเทคนิคดีราคาถูกเลือก BANPU
สำหรับประเด็นต่างประเทศ
ขณะที่ตลาดหุ้นสหรัฐปิดบวกตอบรับข่าวที่สหรัฐเตรียมลดภาษีนำเข้าสินค้าให้กับจีนซึ่งจะช่วยลดความตึงเครียดในเรื่องข้อพิพาทการค้า และจะเป็นปัจจัยที่หนุนตลาดหุ้นทั่วภูมิภาคในวันนี้
โดนัล ทรัมป์ ประธานาธิบดีสหรัฐฯ ประกาศว่าจะไม่เข้าร่วมการประชุม World economic forum ที่จะจัดขึ้นที่เมืองดาวอส ระหว่างวันที่ 22-25 ม.ค. นี้ โดยอ้างเหตุผลว่าเป็นเพราะถูก government shutdown
คุณเทเรซ่า เมย์จะต้องยื่นแผน Brexit ฉบับใหม่ต่อรัฐสภาในวันที่ 21 ม.ค. และรัฐสภาอังกฤษจะทำการอภิปรายและลงมติต่อแผนการแยกตัวของอังกฤษออกจากสหภาพยุโรป (Brexit) ฉบับใหม่ของนางเทเรซา เมย์ นายกรัฐมนตรีอังกฤษ ในวันที่ 29 ม.ค
รายงานประจำเดือน ม.ค. ของ OPEC ระบุว่ากลุ่มประเทศผู้ส่งออกน้ำมัน (โอเปก) กำลังการผลิตน้ำมันของโอเปกลดลง 751,000 บาร์เรล/วันในเดือนธ.ค. สู่ระดับ 31.6 ล้านบาร์เรล/วัน
