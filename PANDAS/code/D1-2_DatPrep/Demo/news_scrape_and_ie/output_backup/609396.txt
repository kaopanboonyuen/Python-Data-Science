หุ้นไทย  เปิดตลาดเพิ่มขึ้น 2.91 จุด นักวิเคราะห์ คาดบวกแคบ หลังวานนี้เผชิญแรงขายดัชนีหลุด 1,800 จุด
บรรยากาศการลงทุนในตลาดหุ้นไทยเช้าวันนี้ (9 ม.ค.61) เปิดทำการเมื่อเวลา 10.00 น. ดัชนีปรับตัวเพิ่มขึ้น 2.91จุด แตะที่ระดับ 1,795.72 จุด ด้วยมูลค่าการซื้อขาย 2,956.24 ล้านบาทโดยหลักทรัพย์เพิ่มขึ้น 279 หลักทรัพย์ ลดลง 210 หลักทรัพย์ ไม่เปลี่ยนแปลง 344 หลักทรัพย์

นักวิเคราะห์ บล.เคจีไอ คาด SET วันอังคารไซด์เวย์บวกแคบ หลังวานนี้เผชิญแรงขายช่วงท้ายตลาด ปิดต่ำกว่าระดับจิตวิทยาที่ 1,800 จุด ทั้งนี้ มอง SET Index เข้าสู่ช่วงพักสร้างฐานแล้ว หลังนักลงทุนต่างชาติยังขายสุทธิต่อเนื่องและแรงสนับสนุนจากนักลงทุนสถาบันในประเทศเริ่มชะลอ ขณะที่ตลาดหุ้นหลักๆ ในต่างประเทศซื้อขายกรอบจำกัดเมื่อคืนนี้ เพื่อรอปัจจัยถัดไปในฝั่งสหรัฐฯ
ทั้งนี้ มองว่าความสนใจของตลาดจะเปลี่ยนจากหุ้นตัวใหญ่ซึ่งรวมพลังดันดัชนีฯ ในช่วงหลายวันที่ผ่านมา มาสู่หุ้นขนาดกลาง-เล็ก อย่างไรก็ตาม หากดัชนีฯ ปรับฐานรอบนี้ น่าจะไม่ลงต่ำกว่า 1,760 จุด ซึ่งจะเป็นระดับที่เปิดอัพไซด์ของหุ้นมากกว่า ณ ปัจจุบัน และเป็นโอกาสในการเก็บหุ้นตัวหลักอีกครั้งหนึ่ง
