แนวโน้มหุ้นไทย พบยังไม่มีปัจจัยบวกใหม่เข้ามากระตุ้น คาดว่าจะแกว่งตัว Sideways พักฐาน เน้นเก็งกำไรหุ้นผลประกอบการแข็งแกร่ง
บริษัทหลักทรัพย์ ฟินันเซีย ไซรัส จำกัด (มหาชน) ระบุในบทวิเคราะห์วันนี้ (24 ก.ค.) ว่า แนวโน้มตลาดหุ้นไทย SET Index คาดว่าจะแกว่งตัว Sideways พักฐาน เนื่องจากไม่มีปัจจัยบวกใหม่เข้ามากระตุ้น นักลงทุนยังคงจับตาดู 
พัฒนาการของประเด็นสงครามการค้าหลังการประชุม G20 ไม่มีข้อสรุป

แต่เน้นย้ำว่าต้องเร่งหารือลดข้อพิพาททางการค้า และโดยเฉพาะอย่างยิ่งการประกาศผลประกอบการ 2Q18 ของบริษัทจดทะเบียนฝั่ง Real Sector ที่จะทยอยออกมาในช่วง 2 สัปดาห์ข้างหน้า ทำให้ระยะนี้เรายังมองว่าหุ้นที่มี 
กำไรโตโดดเด่นจะยังเคลื่อนไหว Outperform ตลาด
สำหรับกลยุทธ์การลงทุน เก็งกำไรหุ้นที่คาดผลประกอบการ 2Q18 แข็งแกร่ง สะสมหุ้นพื้นฐานเพิ่มในช่วงอ่อนตัว ขณะที่หุ้นเด่นเดือน ก.ค. BANPU, CPF, EPG, PTTEP, TISCO
