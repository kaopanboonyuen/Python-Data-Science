หุ้นไทย เปิดตลาดปรับขึ้น 5.31 จุด นักวิเคราะห์ คาด แกว่งตัวเพื่อทดสอบ 1,650 จุด จับตานโยบายทรัมป์ และประชุม FOMC
บรรยากาศการลงทุนในตลาดหุ้นไทยเช้าวันนี้ (12 ก.ย.) เปิดทำการเมื่อเวลา 10.00 น. ดัชนีปรับตัวเพิ่มขึ้น 5.31 จุด แตะที่ระดับ 1,642.85 จุด ด้วยมูลค่าการซื้อขาย 2,161.31 ล้านบาท โดยหลักทรัพย์เพิ่มขึ้น 393 หลักทรัพย์ ลดลง 133 หลักทรัพย์ ไม่เปลี่ยนแปลง 325 หลักทรัพย์
นักวิเคราะห์ บล.เมย์แบงก์ กิมเอ็ง คาด SET แกว่งตัวเพื่อทดสอบ 1,650 จุด โดยใน 2 สัปดาห์ที่ผ่านมา SET ปรับตัวขึ้นโดดเด่นกว่า 4.4% ในขณะที่ภูมิภาคปรับขึ้นเฉลี่ยเพียง 0.5% โดยคาดกลุ่มหลักอย่าง พลังงาน และ ICT จะเริ่มปรับฐาน และอาจมีการ Switch เข้าเก็งกำไรกลุ่มธนาคาร
ทั้งนี้ ยังคงต้องจับตานโยบายของ ทรัมป์ และการประชุม FOMC ที่จะเป็นตัวกำหนดทิศทางตลาดโลก ภายหลังสหรัฐเผชิญกับพายุ Harvey และ ล่าสุด Irma ส่งผลกระทบภาคอุตสาหกรรมพลังงานและการท่องเที่ยวอย่างมาก ทำให้มองว่าสมาชิกสภาพรรค Republican และ Democrat อาจกลับมาช่วยผลักดันมาตราการกระตุ้นเศรษฐกิจต่างๆ ซึ่งอาจส่งผลให้ Dollar กลับมาแข็งค่าอีกครั้ง
