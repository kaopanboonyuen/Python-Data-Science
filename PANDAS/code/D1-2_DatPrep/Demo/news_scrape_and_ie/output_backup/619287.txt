บรรยากาศการลงทุนในตลาดหุ้นไทยเช้าวันนี้ ( 19 ก.พ.) เปิดทำการเมื่อเวลา 10.00 น. ดัชนีปรับตัวเพิ่มขึ้น 8.20 จุด แตะที่ระดับ 1,814.09 จุด ด้วยมูลค่าการซื้อขาย 1,457.89 ล้านบาท โดยหลักทรัพย์เพิ่มขึ้น 322 หลักทรัพย์ ลดลง 144 หลักทรัพย์ ไม่เปลี่ยนแปลง 293 หลักทรัพย์

นักวิเคราะห์ บล.เคจีไอ คาด SET วันจันทร์ขึ้นต่อ แรงขายจากนักลงทุนต่างชาติเริ่มชะลอ ลุ้นจีดีพีไทยไตรมาส 4/2560 สูงกว่า consensus คาด ขณะที่ปัจจัยภายนอกเป็นบวกต่อหลังตัวเลขเศรษฐกิจสหรัฐฯ เดือน ม.ค. ทั้งในส่วนของยอดเริ่มสร้างบ้านและยอดขออนุญาตก่อสร้าง
ด้าน บล.อาร์เอชบี มองว่าตลาดหุ้นไทยวันนี้จะฟื้นตัวได้ในทิศทางเดียวกับตลาดหุ้นในภูมิภาค และตลาดหุ้นสหรัฐฯ ที่มีสัญญาณฟื้นตัวต่อเนื่อง แต่อย่างไรก็ตาม อัตราผลตอบแทนพันธบัตรสหรัฐฯ ที่ทรงตัวอยู่ในระดับสูง ยังกดดันให้ตลาดหุ้นสหรัฐฯ ฟื้นตัวช้า
