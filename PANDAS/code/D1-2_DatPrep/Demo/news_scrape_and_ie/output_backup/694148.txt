บล.ฟินันเซีย ไซรัส ระบุบทวิเคราะห์วันนี้ (18 ธ.ค.) ว่า แนวโน้มตลาดหุ้นไทยเราคาดว่า SET Index จะยังคงแกว่งตัวผันผวน และมีโอกาสอ่อนตัวหลุดระดับ 1,600 จุดอีกครั้งจากบรรยากาศการลงทุนที่เป็นลบ หลังมีความกังวลเรื่องการชะลอตัวของเศรษฐกิจโลก

ส่วนปัจจัยสำคัญที่ต้องติดตามวันนีคือ การกล่าวสุนทรพจน์ของสี จิ้น ผิงในงานฉลองครบรอบ 40 ปีปฏิรูประเทศ รวมถึงการประชุม กนง. และ FOMC ช่วงกลางสัปดาห์ เรามองว่าหุ้น Domestic และ Defensive จะเคลื่อนไหวได้แข็งแรงกว่าตลาดระยะนี้ อย่างไรก็ตาม SET Index บริเวณ 1,585-1,600 ถือว่าไม่แพงทั้งในแง่ Valuation และ Earning Yield Gap ที่กว้างขึ้นเป็น4.5% ซึ่งปกติมักจะเกิดการรีบาวด์
สำหรับกลยุทธ์การลงทุนหุ้นไทย แนะสะสมหุ้นและคาดหวังการรีบาวด์บริเวณ 1,585-1,600 จุด โดยเน้น Domestic และ Defensive Play หุ้นเด่นเดือนธ.ค. : ADVANC, AOT, BCH, CK, SISB
