นักวิเคราะห์ บล.เคจีไอ คาด SET วันพฤหัสฯ ไซด์เวย์ ลบกรอบจำกัด โดยมองว่าภาพใหญ่คือหุ้นเข้าช่วงพักตัวแล้ว ทั้งนี้ยอดขายสุทธิที่ค่อนข้างสูงจากต่างชาติเมื่อวานนี้ ผนวกกับแรงซื้อจากฝั่งสถาบันฯ ที่ชะลอ น่าจะกระทบจิตวิทยาของ SET ในระยะสั้น

ขณะที่ปัจจัยภายนอกเป็นลบเล็กน้อย ตลาดหุ้นสหรัฐฯ ปรับลดลงเป็นครั้งแรกของปี 2561 หลังบลูมเบิร์กรายงานว่า เจ้าหน้าที่ของทางการจีนแถลงว่าอาจมีการชะลอการซื้อพันธบัตรรัฐบาลสหรัฐฯ หลังความสัมพันธ์ระหว่างประเทศเป็นลบมากขึ้นในช่วงที่ผ่านมา รวมถึงมีแนวโน้มที่สหรัฐฯ จะยกเลิกเขตการค้าเสรีนาฟตากับแคนาดาและเม็กซิโก
อย่างไรก็ดี ฝ่ายวิจัย KGI มองว่าข่าวสารดังกล่าวไม่ได้เชื่อมโยงกับหุ้นไทยมากนัก ผนวกกับพื้นฐานภายในประเทศแข็งแกร่งน่าจะจำกัดดาวน์ไซด์ของ SET ได้อย่างดี อย่างไรก็ตาม วานนี้รองนายกฯ ด้านเศรษฐกิจ คาดจีดีพีไตรมาส 4/2560 โต 5.0% ใกล้เคียงกับประมาณการของนักเศรษฐศาสตร์ KGI
และล่าสุด ก.คลังเตรียมเสนองบกลางเพิ่มเติม 1 แสนล้านบาท เข้า สนช. เดือน มี.ค. นี้ หวังกระตุ้นสวัสดิการรัฐฯ ปฎิรูปภาคเกษตร และพัฒนาเศรษฐกิจฐานราก น่าจะเป็นบวกต่อแนวโน้มจีดีพีเพิ่มเติมอีก
