บล.ฟินันเซีย ไซรัส ระบุบทวิเคราะห์วันนี้ (25 พ.ค.) ว่า แนวโน้มตลาดหุ้นไทยคาดวํา SET Index จะยังแกว่งตัว Sideways Down จากบรรยากาศการลงทุนที่ค่อนข้างเป็นลบ หลังล่าสุดสหรัฐฯ ยกเลิกการพบเกาหลีเหนือกลางเดือนหน้า ขณะที่ราคาน้ำมันดิบปรับลงซึ่งทำให้กลุ่มพลังงานยังคงถูกกดดันนอกเหนือจากปัจจัยในประเทศเรื่องการตรึง 
ราคาดีเซลและแก๊ส

ขณะที่สัปดาห์หน้ามีปัจจัยสำคัญต้องติดตามการวินิจฉัยของศาลรัฐธรรมนูญเรื่องกฎหมายเลือกตั้ง ส.ส. ในวันที่ 30 พ.ค.การฟื้นตัวของ SET ในระยะนี้จึงยังดูจำกัด เราประเมินกรอบล่างของตลาดบริเวณ 1,720 จุด หรือแย่ที่สุดคือ 1,700 จุด
สำหรับกลยุทธ์การลงทุน เก็งกำไรหุ้น Laggard และ Defensive สะสมหุ้นพื้นฐานในช่วงพักตัว หุ้นเด่นเดือน พ.ค. BEM, CHG, EA, SC, THANI
