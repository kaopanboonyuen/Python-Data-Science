เตรียมให้บริการ LINE TAXI สิ้นปี 2017 ยัน สะดวกปลอดภัย แก้ปัญหาการปฏิเสธผู้โดยสาร
วันที่ 31 ก.ค. 2560 ที่โรงแรม ดับเบิ้ลยู กรุงเทพ นายสนิท พรหมวงษ์ อธิบดีกรมการขนส่งทางบก เป็นประธานเปิดโครงการ THAI TAXI 4.0 โดยมีความร่วมมือจาก Line และเครือข่ายสหกรณ์แท็กซี่เขตกรุงเทพมหานคร ร่วมกันเซ็นสัญญาพัฒนาบริการ LINE TAXI ภายใต้แนวคิด TAXI 4.0

โดยนำความแข็งแกร่งของ LINE ด้านจำนวนผู้ใช้ผนวกกับเครือข่ายคนขับแท็กซี่ถูกกฎหมายกว่า 60,000 ราย มาพัฒนารูปแบบการให้บริการที่ง่ายและสะดวกยิ่งขึ้นโดยผ่านแอพพลิเคชั่น LINE ซึ่งจะทำให้ผู้โดยสารไม่ต้องเสียเวลารอนาน อีกทั้งยังปลอดภัย โดย LINE TAXI โดยขณะนี้อยู่ในขั้นตอนการพัฒนาให้สมบูรณ์ และจะเริ่มให้บริการภายในสิ้นปี 2017
นายอริยะ พนมยงค์ กรรมการผู้จัดการ LINE ประเทศไทย กล่าวว่า จุดเด่นของ LINE TAXI คือการนำความแข็งแกร่งของ LINE ที่มีผู้ใช้งานครอบคลุมกว่า 90% ของผู้ใช้งานอินเทอร์เน็ตบนมือถือในประเทศไทย มาผนึกกำลังกับเครือข่ายแท็กซี่เขตกรุงเทพมหานคร ซึ่งปัจจุบันผู้บริโภคต้องการขีวิตที่รวดเร็ว สะดวกสบาย และปลอดภัย เราจึงพัฒนาบริการดังกล่าวเพื่อตอบโจทย์การใช้ชีวิตของคนไทย

นายวิฑูรย์ แนวพานิช ประธานเครือข่ายสหกรณ์แท็กซี่เขตกรุงเทพมหานคร กล่าวว่า ที่ผ่านมาปัญหาหลักที่ส่วผลกระทบต่อภาพลักษณ์ของแท็กซี่คือการปฏิเสธผู้โดยสารซึ่งมีสาเหตุเกิดจากรายได้ของคนขับที่ไม่คุ้มทุนในแต่ละวัน ทั้งนี้ เทคโนโลยีดังกล่าวจะเข้ามาช่วยแก้ปัญหาได้
ซึ่งบริการแท็กซี่รูปแบบนี้จะช่วยสร้างมาตรฐานด้านคุณภาพการให้บริการที่สูงขึ้น และช่วยเพิ่มโอกาสในการเรียกใช้งานของผู้โดยสาร และตามมาซึ่งรายได้และความเป็นอยู่ที่ดีขึ้นของแท็กซี่ นอกจากนี้ยังถือเป็นก้าวใหม่สู่โลกยุคดิจิทัลของแท็กซี่ไทยอย่างเต็มรูปแบบ และผู้โดยสารจะได้รับบริการที่ปลอดภัย ประทับใจ และเชื่อถือได้ ไม่ถูกปฏิเสธอย่างแน่นอน
