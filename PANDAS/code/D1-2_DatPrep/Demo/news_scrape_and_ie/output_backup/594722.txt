ยอดจำหน่าย  แสตมป์  ชุดพระราชพิธีถวายพระเพลิง วันแรกกว่า 2.4 ล้านชุด สามารถหาได้ที่ทำการไปรษณีย์จนกว่าสินค้าจะหมด
บริษัท ไปรษณีย์ไทย จำกัด (ปณท) เผย ตราไปรษณียากรที่ระลึกพระราชพิธีถวายพระเพลิงพระบรมศพ พระบาทสมเด็จพระปรมินทรมหาภูมิพลอดุลยเดช บรมนาถบพิตร ที่จัดจำหน่ายวันนี้วันแรก อยู่ที่ 2,400,000 ชุด จากยอดการพิมพ์ 3,000,000 ชุด หรือคิดเป็นยอดการจำหน่ายไปแล้วกว่า 80 เปอร์เซ็นต์
โดยจำนวนดังกล่าว เป็นจำนวนรวมจากการเปิดจำหน่ายตราไปรษณียากรชุดที่ระลึกในวันแรก และยอดจองเดิมที่เปิดให้จองตั้งแต่วันที่ 28 สิงหาคม 2560 ที่ผ่านมา
ทั้งนี้ ผู้สนใจยังสามารถหาซื้อแสตมป์ชุดดังกล่าวได้ ณ ที่ทำการไปรษณีย์ทั่วประเทศ ในราคาชุดละ 99 บาท จนกว่าสินค้าจะหมด สอบถามรายละเอียดเพิ่มเติมได้ที่ ฝ่ายตลาดตราไปรษณียากร 0 2573 5480, 0 2573 5463, 0 2831 3856 facebook : stampinlove และไลน์ @stampinlove
